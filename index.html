Container

Help you to starting your web layout, Container .container is classes that adds 16px left and right padding to your HTML tag. The default of Container width is 80% (on 768px, 992px and more than 1200px) and 100% (on less than 768px). You can use additional class of container, full .container.full to create Full Width Container.

Default Container

<div class="container">
  This is Default Container
</div>

Full Width Container

<div class="container full">
  This is Full Width Container
</div>

Grid

12 Column responsive grid view. The columns will re-arrange automatically dependeing screen size. On large screen, it might look inline according to column size and on small screen, it's might look like card stack. When on small screen, grid childs (.lg-*, .md-*, .sm-*) add 16px margin bottom.

<div class="grid">
  <div class="set lg-6 md-8 sm-4">
    50% on large, 66% on medium 33% on small
  </div>
  <div class="set lg-6 md-4 sm-8">
    50% on large, 66% on medium 33% on small
  </div>
</div>
50% on large, 66% on medium 33% on small
50% on large, 33% on medium, 66% on small

Classes

Class Description
grid Container for responsive grid view
set Defines a column and adds 16px padding on right and left.
xl-* Defines column on extra large screens (1200px and up).
lg-* Defines column on large screen, typical laptops and desktops (992px and up).
md-* Defines column on medium screen, typical tablets (768px and up).
sm-* Defines column on small screen, typical smartphones (576px and up).
off-xl-*, off-lg-*, off-md-*, off-sm-* Move columns to right with adds left margin to column.

Offset

Push the column to the right. When the screen size changes, the offset will change according to the class offset. If in a column there is only 1 class of offset (for example only off-lg-*) then if the screen size changes to medium or lower, the offset will disappear.

Resize to see effect.

33% on large screen and margin left 66%
33% on large screen and offset 33%
33% on large screen and no offset.

<div class="grid">
  <div class="set lg-4 off-lg-8">
    33% on large screen and margin left 66% on large screen
  </div>
</div>
<div class="grid">
  <div class="set lg-4 off-lg-4">
    33% on large screen and margin left 33% on large screen
  </div>
</div>
<div class="grid">
  <div class="set lg-4">
    33% on large screen and no margin left on large screen
  </div>
</div>
